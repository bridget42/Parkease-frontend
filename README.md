# ParkEase Frontend

# Project Description
ParkEase is a comprehensive parking management system that allows attendants to register vehicles, process check-outs, and generate receipts. The admin dashboard provides revenue reports and vehicle tracking capabilities.

# Pages
- **Login Page** - Authentication gateway with role-based redirect
- **Attendant Dashboard** - Overview of parked vehicles, revenue stats, and quick actions
- **Vehicle Registration** - Form for capturing driver and vehicle information with validation
- **Sign-out & Receipting** - Vehicle checkout with fee calculation and receipt generation
- **Admin Reports** - Revenue analytics with filtering and pagination
